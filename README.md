# Орфоэпический тренажёр | 5 класс

Интерактивная игра-тренажёр для отработки навыков постановки ударения в русском языке.

## 🎮 Функционал

- Ввод имени игрока
- 20 случайных слов за раунд (из базы 68 слов)
- Два варианта ударения на выбор
- Визуальная обратная связь с анимациями и эмодзи
- Разбор ошибок с отображением правильного ответа
- Таблица лидеров с лучшими результатами за каждый день
- Адаптивный дизайн для телефонов и компьютеров

## 🎯 Как играть

1. Введите ваше имя
2. Нажмите "Начать игру"
3. Выберите правильный вариант ударения из двух предложенных
4. Нажмите "Проверить" — узнаете, верно ли ответили
5. Если не уверены — нажмите "Пропустить"
6. После 20 слов откроется разбор ошибок
7. Нажмите "Таблица рекордов", чтобы увидеть свои лучшие результаты

## 🎨 Дизайн

- Цветовая палитра: зелёная (различные оттенки зелёного)
- Адаптивный интерфейс для всех устройств
- Анимации при правильных/неправильных ответах
- Эмодзи для эмоциональной обратной связи

## 📊 Особенности

- Умное повторение: слова с ошибками возвращаются в следующих раундах
- Хранение всех результатов в localStorage
- Показ только лучших результатов за каждый день
- Автоматическая очистка старых данных (30+ дней)
- Уникальные слова в каждом раунде (выборка без повторов)
- Детальный разбор ошибок с указанием номера слова

## 📁 Структура проекта

- `index.html` — основной HTML файл
- `style.css` — стили и адаптивный дизайн
- `script.js` — логика игры, подсчёт очков, таблица рекордов
- `words.js` — база слов с правильным и неправильным вариантами ударения

## 🚀 Запуск

Просто откройте файл `index.html` в любом современном браузере. Никаких дополнительных зависимостей и серверов не требуется.

## 📝 Правила игры

- За каждый правильный ответ начисляется +1 к счёту
- За ошибку или пропуск +1 к ошибкам
- Пропущенное слово засчитывается как ошибка
- На 20-м слове игра завершается автоматически после ответа
- Результаты сохраняются только за текущий день
- Таблица показывает топ-5 лучших результатов (по проценту правильных ответов и времени)
